Implementation of audio signal augmentation. 

Types of augmentation: changing the speed of the audio signal, adding white noise, adding reverberation, removing a frequency section.

Launching the solution:
1) Installing the required versions of the pip install libraries requirements.txt
2) Audio examples (mono and stereo) in the audio_examples folder.
3) Launching the CLI_app solution.
4) The result of the solution will be saved in the augmented folder.
